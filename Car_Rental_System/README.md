# 🚗 Car Rental System (Python OOP Project)

This project is a **Command-Line Car Rental System** built using **Python** to practice **Object-Oriented Programming (OOP)** concepts.

The system simulates a simple car rental service where users can manage cars, register customers, rent vehicles, and return them.

---

## 📌 Features

* Add new cars
* Register customers
* Show available cars
* Show registered customers
* Rent cars for a number of days
* Return rented cars
* Automatic ID generation for cars and customers
* Rental price calculation

---

## 🧠 Concepts Used

This project demonstrates several **Python OOP concepts**:

* Classes and Objects
* Encapsulation
* Composition (Rental contains Car + Customer)
* Class Variables for unique ID generation
* Menu-driven CLI application
